GRAPPLING

Install these 2 Siri Shortcuts. Each takes 30 seconds to set up.

▶ Build "Status" Shortcut

Opens Shortcuts app → New shortcut → Add these actions:
1. Get contents of URL:
raw.githubusercontent.com/aarontmaher/Chat-gpt/main/results.md
2. Get text between markers
3. Speak text
Name it: Status

▶ Build "What's Next" Shortcut

Same steps but URL:
raw.githubusercontent.com/aarontmaher/Chat-gpt/main/CLAUDE.md
Get text between: ## PENDING TASKS and ## LOCKED
Name it: What's Next

Say: "Hey Siri Status" or "Hey Siri What's Next"
